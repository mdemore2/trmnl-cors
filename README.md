# trmnl-cors

A passthrough server to poll APIs and present data to Terminus without CORS

## Usage

### Environment Variables

`WIKI_API_TOKEN`, `WIKI_API_APP_NAME`, `WEATHER_API_KEY`, `WEATHER_ZIP_CODE`, `SURFLILNE_SPOT_ID`, `SURFLINE_BUOY_ID`
